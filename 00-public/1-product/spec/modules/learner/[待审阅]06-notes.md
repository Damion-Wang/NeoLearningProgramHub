# 笔记模块

## 模块定位

笔记是学员在学习全程中随时记录想法的工具——以悬浮窗/悬浮球形态存在，独立于授课流程，本期不与课程内容打通。

### 关联关系

笔记贯穿学员端所有页面，但使用场景和关联程度不同：

| 关联页面 | 关联方式 | 典型场景 |
|---------|---------|---------|
| **大厅（01-hall）** | 笔记入口常驻，可查看/编辑所有笔记 | 学员在辅导对话中记录教练的建议；回顾之前课程中记的笔记 |
| **授课教室（02-teaching）** | 笔记悬浮球常驻，随时展开记录 | AI讲解PIN法时记下关键要点；做Quiz时粘贴自己的思考 |
| **对练教室（03-drill）** | 笔记悬浮球常驻，对练中/复盘后均可记录 | 复盘环节记录AI的反馈要点；记下自己想改进的表达方式 |
| **测评教室（04-assessment）** | 笔记悬浮球常驻，测评后记录 | 信息确认复盘时记录自己的能力特点 |
| **报告教室（05-report）** | 笔记悬浮球常驻，看报告时记录 | 记下报告中的行动建议；截图保存关键图表 |

**本期限制**：笔记内容与课程内容相互独立，不自动关联。未来可能打通——如在笔记中自动标注"此笔记记录于PIN法课程第3页"。

---

## 呈现形态

笔记不占用页面布局——以独立悬浮层形态存在。

| 形态 | 说明 |
|------|------|
| **悬浮球**（默认） | 屏幕边缘的小圆球图标，随时可点击展开为悬浮窗 |
| **悬浮窗**（展开后） | 可拖拽移动的浮窗，大小可调，可最小化回悬浮球 |

## 用户故事：张磊在授课中记笔记

### 场景背景
张磊正在授课教室学习 PIN 法。AI 讲到 Interest 层时，他突然想到一个自己工作中的关联场景，想记录下来。

### 完整旅程

**打开笔记**

屏幕右下角有一个小的悬浮球图标（笔记标志）。张磊点击悬浮球 → 展开为悬浮窗。

悬浮窗浮在授课界面之上，不影响授课进度——AI 继续讲课，张磊可以边听边记。

**记录内容**

悬浮窗中是一个简洁的编辑器，张磊输入：
> "Interest 这个概念让我想到——上次张总找我要人，他的 Interest 可能不是'要人'，而是'活动不能出错'。下次再遇到这种情况，先问'你最担心的是什么'。"

张磊还想记录当前 PPT 上的冰山模型图 → 他用系统截图功能截取了 PPT 画面 → 粘贴到笔记中。

**保存笔记**

张磊点击「保存」→ 笔记自动保存。他点击悬浮窗的最小化按钮 → 窗口缩回悬浮球，继续听课。

**在大厅回看笔记**

学完课程回到大厅后，张磊点击悬浮球打开笔记 → 看到之前记录的所有笔记（按时间排列）。

他还可以在大厅的中间看板「笔记」区域查看笔记列表（笔记的另一个访问入口）。

## 核心交互清单

### 1. 打开笔记

| 项目 | 内容 |
|------|------|
| **触发** | 学员点击屏幕边缘的悬浮球图标 |
| **响应** | 悬浮球展开为悬浮窗，浮在当前界面之上 |
| **联动** | 不中断当前任何活动（授课继续、对练继续） |

### 2. 编辑笔记

| 项目 | 内容 |
|------|------|
| **触发** | 学员在悬浮窗内操作 |
| **响应** | 支持：打字输入、文字编辑、粘贴文本、插入截图、保存 |
| **联动** | 笔记自动保存，不需要手动触发 |

### 3. 最小化笔记

| 项目 | 内容 |
|------|------|
| **触发** | 学员点击悬浮窗的最小化按钮 |
| **响应** | 悬浮窗缩回悬浮球，回到屏幕边缘 |
| **联动** | 笔记内容自动保存，下次打开时恢复 |

### 4. 拖拽移动

| 项目 | 内容 |
|------|------|
| **触发** | 学员拖拽悬浮窗标题栏 |
| **响应** | 悬浮窗跟随鼠标移动，可放置在屏幕任意位置 |
| **联动** | 悬浮球也可以拖拽到屏幕边缘任意位置 |

### 5. 大厅笔记入口

| 项目 | 内容 |
|------|------|
| **触发** | 学员在大厅中间看板点击「笔记」区域 |
| **响应** | 展示笔记列表（按时间倒序排列），可查看/编辑/删除 |
| **联动** | 与悬浮窗中的笔记是同一份数据 |

## 已确认决策

- 笔记改为悬浮窗/悬浮球形态——独立于授课，不在左侧看板中
- 笔记和授课是独立的两条线
- 本期功能：打字、编辑、粘贴、插入截图、保存
- 本期不与授课内容打通——不做笔记关联课程时间点、不做 AI 自动笔记
- 笔记可在大厅看板中查看（额外的访问入口）

---

## 本期不做的功能

- 笔记与授课内容打通——不关联课程时间点，不在笔记中跳转到课程位置
- AI 自动生成笔记——本期纯手动记录
- 笔记导出——本期不支持导出 PDF/文档
- 笔记分享——不支持学员间分享笔记
- 笔记标签/分类——本期不做分类体系
