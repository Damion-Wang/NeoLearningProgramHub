## 关键Debate修订总结

### P1修订（全部应用）
1. 比例切换重置规则（同类型保持/不同类型重置）
2. 联动防抖（500ms debounce + 后到取消前）
3. 回顾状态嵌套中断规则
4. Quiz提交失败回退状态
5. "重新开始"范围与进度处理
6. 断点恢复->回大厅使用引用消息变体
7. 对练三阶段UI切换分步串联
8. 角色图预生成缓存
9. 内容安全延迟预算+流式输出目标值
10. 断点续播分阶段恢复策略
11. 图表点击解读取消前序
12. BI图表MVP缩减为3种
13. REF标注后端解析+降级
14. 催学协商完整流程
15. HR端催学规则全览
16. 双角色入口/切换
17. Agent权限隔离方案
18. 管理端降级/异常方案
19. HR端数据导出
20. 报告版本关系明确

### 重要P2修订（已应用）
- HR/运营端比例调整为15:60:25
- 首次进入大厅看板特殊态
- 看板模块默认折叠规则
- 断点弹窗时间阈值(<5min不弹窗)
- 对话区首条不重复Quiz题干
- 分位标尺可配置展示
- 导航命名统一
- HR/运营端视图模式标注

- 视频暂停状态下提问处理细节
