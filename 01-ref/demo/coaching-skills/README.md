# coaching-skills/ -- 辅导模块 Demo 项目

此目录包含辅导模块的完整原型实现（JavaScript 项目），是项目技术参考中最重要的素材之一。

## 目录结构

```
coaching-skills/
└── coaching-skills/    项目主目录（含完整代码和设计文档）
    ├── docs/           设计文档（Agent系统规格、Leo全景、私教系统规格）
    ├── personas/       Persona 层（Coach/Lecturer 双人格）
    ├── engines/        认知引擎层（人格、提问、BEI、能力追踪等7个引擎）
    ├── skills/         Skill 层（12个技能）
    ├── quality-inspection/  质量检验层（规则+教学法+LLM三层质检）
    ├── data/           教练知识数据
    ├── methodology/    方法论参考
    └── todo/           开发待办
```
