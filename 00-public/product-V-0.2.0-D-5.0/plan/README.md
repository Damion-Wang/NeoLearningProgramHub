## 关键产品决策（v0.5.0 已锁定）

- **品牌命名**：睿学（NeoLearning）/ 本期交付物 = NeoLearningProgram
- **三大底层原则**：第一性 / 懂你 / 像人
- **学员侧场域**：本期 3（lecture / practice / recap）+ 未来（inquiry / assignment / 沙盘 等）
- **三类账号**：学员 / 管理员 / admin（项目根账号）
- **三个端口**：学员端（产品化）/ 管理端（产品化）/ 产品运营端（短期非产品化）
- **服务周期**：开营前 admin 可改 / 开营后锁定 / 默认 1 年
- **项目周期**：默认 = 服务周期 - 1 周 / 开营后可调（严格确认）
- **学员上限**：本期固定 100；签单即锁定
- **team report 是核心交付引擎**：本期"综合报告"模板
- **数据交付分层**：管理端产物可导出 / 原始学习记录不导出（防反推）
