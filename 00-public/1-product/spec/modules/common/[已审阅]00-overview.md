# 通用模块总览

## 定位

通用模块是跨端口、跨页面的基础功能集合。不属于学员端教室，也不属于管理端看板，而是所有页面共享的通用能力。全部纯 GUI，不涉及 Agent。

---

## 通用模块清单

| # | 模块 | 入口 | 子文档 |
|---|------|------|--------|
| 1 | **通用功能**（账号管理+个人设置+帮助） | topbar → 头像下拉 → 各子项弹窗打开 | [01-general.md](01-general.md) |
| 2 | **消息提醒** | topbar → 铃铛🔔下拉面板 | [02-notification.md](02-notification.md) |

## 顶部通用栏（Topbar）设计

### 布局

```
┌──────────────────────────────────────────────────────────┐
│ [Logo] [当前端口名称]                     [🔔铃铛] [👤头像] │
└──────────────────────────────────────────────────────────┘
```

- **左侧**：产品 Logo + 当前端口标识（"学员端" / "HR端" / "运营端"）
- **右侧**：两个可交互元素

### 铃铛🔔 — 消息提醒入口

- 有未读消息时显示红点
- 点击后下拉消息面板（消息列表 + 右上角"全部已读"按钮）
- 本期消息只读，不支持回复
- 三类消息：平台消息 / 系统消息（自动催学）/ 运营消息（群发或定点）
- 详见 [02-notification.md](02-notification.md)

### 头像👤 — 个人功能入口

点击后展开下拉菜单：

| 菜单项 | 交互 | 说明 |
|--------|------|------|
| **账号管理** | 弹窗 | 个人信息查看编辑 + 密码修改 |
| **个人设置** | 弹窗 | AI语速（本期）+ 语种/页面风格/个人偏好（待上线） |
| **帮助** | 弹窗 | 简单静态 FAQ + 联系方式 |
| **端口切换** | 展开子下拉列表 | 列出当前账号可用的端口（学员端/HR端/运营端），选择后页面切换 |
| **退出登录** | 弹窗确认 | "确定退出登录？" → 确认后回到登录页 |

账号管理、个人设置、帮助三个页面详见 [01-general.md](01-general.md)

### 端口切换细节

- 点击"端口切换"后展开子下拉列表
- 列出当前账号拥有的所有角色对应的端口
- 当前端口高亮标识，其余可点击切换
- 只有学员角色的用户看不到"HR端""运营端"选项
- 切换后页面刷新为目标端口的默认页面
- HR 切换为学员端时，若账号不在学员名单中，体验数据单独记录

### 三端差异

通用栏 UI 在三端完全一致，差异仅在于：
- **端口切换的可选项**：根据账号角色动态显示
- **消息内容**：学员收学习提醒，HR/运营收催学和进度相关消息

## 通用栏 vs 管理端导航

| 层级 | 位置 | 功能范围 |
|------|------|---------|
| **通用栏**（顶部） | 所有页面最顶部 | 个人级操作（账号/设置/消息/切换/退出） |
| **管理端导航**（左侧） | 仅管理端页面 | 项目级操作（看板/报告/项目配置） |

---

## 已确认决策

- 通用栏三端完全一致
- 头像下拉除退出登录弹窗外，其余跳转独立GUI页面
- 消息本期只读不支持回复
- 端口切换用子下拉列表
- 全部纯 GUI，不涉及 Agent
