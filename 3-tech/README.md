# 3-tech/ -- 技术分析与设计

## 技术现状

Prosona Agent 是当前已有的 AI 教学引擎，基于 LangGraph 构建。

**已实现：**
- 授课引擎（AOM 驱动，NARRATION + CHAT 两种交互）
- 对练引擎（导学→演绎→复盘→报告四阶段，导演-演员分离）
- 个人报告（独立子图，8大模块并行生成）
- L2 自适应（separateTag 分流机制）
- ACE 上下文工程（设计完成，部分落地）

**未实现（但有参考）：**
- 辅导模块 — **coaching-skills demo 已有完整原型**（4层架构、12个Skill、贝叶斯能力追踪、三层质检）
- 独立测评模块（BEI/量表/学员画像）— coaching-skills 中有 BEI 引擎和能力框架原型
- 记忆工程三级管道（ACE 已设计，管道未落地）
- 管理端（HR/运营的所有功能）

## 可复用的设计模式（来自 coaching-skills demo）

1. **Persona-Skill 分层** — 人格和技能解耦，同一 Skill 可被不同 Persona 组合
2. **动态 Prompt 组装** — 11 板块 System Prompt 按上下文动态拼接
3. **LLM 降级双路径** — LLM-first + 规则降级，保证可用性
4. **认知链提问约束** — 苏格拉底式追问有结构化约束，不是自由发挥
5. **贝叶斯能力追踪** — Beta 分布 + 6 重抗刷分，能力评估有数学基础
6. **三层质检管线** — 规则扫描→教学法合规→LLM 深度审查
7. **BEI STAR+DSTF 模型** — 行为事件访谈的结构化框架
8. **信号驱动路由** — 学员信号触发教学策略切换
9. **教练范例库** — 标准化的优秀教练回复样例
