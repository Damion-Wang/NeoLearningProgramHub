# ADR-003: 辅导模块作为学员端主界面/入口

**状态：** 已决定
**日期：** 2026-04-10
**决策者：** 产品负责人

## 背景

学员端需要一个主界面。可以是课程列表、任务看板、或对话式界面。

## 决策

辅导模块（1v1 AI 教练对话）作为学员端主界面。左侧看板（进度/课程结构/笔记/资产/待办），右侧 Chat UI。教学模块全屏切换进入，可随时返回辅导。

## 理由

- 对话式交互最符合"睿学"的产品定位
- 辅导教练可以主动推荐学习任务，引导学员进入教学模块
- 长期陪伴的 1v1 关系增强学员粘性
- 左侧看板提供结构化信息，补足纯对话的不足

## 后果

- 正面：体验差异化极强，与传统 LMS 完全不同
- 负面：Chat UI 对部分用户可能有学习成本；需要辅导 Agent 的质量足够高
