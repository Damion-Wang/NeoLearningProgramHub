# learner — 学员端模块

> 学员使用睿学产品的全部界面。大厅 + 四间场域 + 笔记 + 发现库。

## AI 角色

- **Neo**（统一）：模拟 1v1 线下授课老师，大厅展现 Coach 能力（陪伴/辅导/Bloom 引导/情绪感知），四间场域展现 Tutor 能力（结构化教学/AOM 脚本）
- **Actor**（演员）：对练场域角色扮演
- **导演**（后台）：对练场域指挥 Actor

> **Neo 第一性原理**：模拟 1v1 线下授课老师，通过讲授/解惑/讨论/考研/练习/追问/反思 7 个动作，逼着或带领学生掌握内容。
