## 主要变更（D-5.0 → D-5.1）

1. **状态机简化** — 4 学员状态（空白/部分/完整/自由复习）→ 3 状态（锁定/完整态首聊/自由复习）
2. **报告结构重组** — 完成事件层 5 块 → 报告 5 章新结构（综合评估 / 学了什么 / 表现怎么样 / 学得怎么样 / 接下来）
3. **变化叙事模型** — 引入"关联 Activity 集合 起点 vs 终点"对比 / 文字叙事 + 对话片段举证（不量化）
4. **行动建议便签化** — 立即/近期/中期模糊时间 / Neo 任务卡片同步推送（学员选择接受） / 删推荐学习卡片
5. **知识关联文字化** — if-then 应用关联文字段（60-100 字）替代流线/树状关联图 / Neo 实时推导（不依赖 KGP）
6. **Neo 双层语气** — 报告语气=数据显示/观察到 / 场域语气=我（1v1 老师）
7. **低数据兜底** — 综合评估章节降级机制 / 各章节兜底文案
