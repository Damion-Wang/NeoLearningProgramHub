# learner — 学员端模块

> 学员使用 AI 老师产品的全部界面。大厅 + 四间场域 + 笔记。

## AI 角色

- **Leo**（Coach）：大厅 1v1 成长伙伴
- **Neo**（Tutor）：四间场域结构化教学
- **Actor**（演员）：对练场域角色扮演
- **导演**（后台）：对练场域指挥 Actor
