## 当前辩论机制（v2.0）

主持人 + 挑战者 + 主题相关角色（3-5 个，按主题选）。**5 Phases 流程**：
1. Phase 0 · 底层共识对齐（debate plan 之前 多轮 Q&A 拉齐 DM 与主持人）
2. Phase 1 · 信息预备（项目内必读 + 网络必搜 预先准备）
3. Phase 2 · debate plan（议题/角色/节奏 8-12 题确认）
4. Phase 3 · 每轮 4 子环节（轮流 / 单挑 / 自由辩论 / 裁定）
5. Phase 4 · 轮间环节（调研 / DM 互动 / 结论 / DM 总结 5 件套记录）
