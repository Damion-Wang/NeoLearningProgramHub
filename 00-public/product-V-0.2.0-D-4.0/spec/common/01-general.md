# 通用功能：账号管理 + 个人设置 + 帮助

## 模块定位

头像下拉菜单中的三个功能——账号管理、个人设置、帮助。点击菜单项后以**弹窗**形式打开，不跳离当前页面。全部纯 GUI，不需要 Agent。

---

## 一、账号管理页面

### 功能

| 功能 | 说明 |
|------|------|
| **个人信息查看** | 展示姓名、头像、手机号、邮箱等基本资料 |
| **个人信息编辑** | 支持修改姓名、上传/更换头像、绑定/更换手机号和邮箱 |
| **密码修改** | 输入旧密码 → 输入新密码 → 确认新密码 → 保存 |

### 用户故事

> **注意**：以下用户故事仅供参考，账号管理的具体交互以 GPB 对接需求为准。

**张磊想修改自己的头像和密码**

1. 张磊点击 topbar 右侧头像 → 下拉菜单 → 点击"账号管理"
2. 弹窗打开账号管理，展示个人信息表单
3. 张磊点击头像区域的"更换"按钮，上传一张新照片，裁剪后保存
4. 张磊点击"修改密码"，输入旧密码和新密码，点击保存
5. 系统提示"密码修改成功"，下次登录生效
6. 张磊点击页面左上角返回按钮，回到之前的页面

### 已确认决策

- 本期不支持手机号自助注册（所有账号由运营导入创建）
- 对接 GPB 账户体系（登录验证由 GPB 处理）

## 二、个人设置页面

### 功能

| 功能 | 状态 | 说明 |
|------|------|------|
| **AI 语速** | ✅ 本期 | 三挡选择：**舒适**（偏慢）/ **标准**（默认）/ **明快**（偏快）。此处为全局设置，对所有场景生效。各场域（如授课场域）内可局部调整语速，局部修改仅作用于该场域；全局修改会重置所有局部设置。AI 音色不变 |
| **页面风格** | ✅ 本期（如有余力尽量上线，之前已有基础） | 浅色/深色模式切换 |
| **语种** | 🔒 待上线（置灰占位） | 界面语言切换 |
| **个人偏好** | 🔒 待上线（置灰占位） | 其他个性化偏好 |

### 用户故事

**张磊觉得 睿学说话太慢，想加速**

1. 张磊点击 topbar 头像 → 下拉菜单 → 点击"个人设置"
2. 弹窗打开个人设置
3. 张磊看到 AI 语速三个选项：舒适 / **标准**（当前选中）/ 明快
4. 他点击"明快"，页面显示"已保存"
5. 返回授课场域，睿学的语速变快了，但音色没变
6. 张磊注意到下方有"语种""页面风格""个人偏好"三个灰色入口，标注"即将上线"

### 关键设计

- **全局设置在个人设置页面**，调整后对所有场景生效（授课/对练/辅导/调研）
- **场域内可局部覆盖**——各场域内可单独调整语速（如授课场域内改为"舒适"），局部修改仅作用于该场域
- **全局重置局部**——当全局设置被修改时，会重置所有场域的局部设置为新的全局值
- 语速挡位命名模拟真人语气感受（舒适/标准/明快），不使用倍速标签（学员界面不显示 1.0x 等数字）

## 三、帮助（产品使用手册）

### 功能

| 功能 | 说明 |
|------|------|
| **产品使用手册** | 一系列文档形式的使用指南，覆盖各功能模块的操作说明（静态内容，本期手动维护） |
| **联系方式** | 客服/技术支持的联系方式 |

### 交互方式

- 点击"帮助"后**打开新窗口**（不是弹窗），展示完整的产品使用手册
- 手册以文档列表形式组织，学员可按模块浏览

### 用户故事

**王芳不知道怎么查看自己的学习报告**

1. 王芳点击 topbar 头像 → 下拉菜单 → 点击"帮助"
2. 浏览器打开新窗口，展示产品使用手册目录
3. 她在目录中找到"如何查看个人学习报告？"，点击进入该文档
4. 文档说明了从大厅进入报告场域的路径，配有截图和步骤
5. 王芳关闭帮助窗口，回到之前的页面继续操作

### 本期范围

- 静态文档，内容由运营手动维护
- 不做搜索功能
- 不做智能客服/AI 帮助

---

## 已确认决策（整体）

- 三个页面全部纯 GUI，无 Agent
- 账号管理、个人设置以弹窗打开，关闭弹窗回到当前页面；帮助以新窗口打开
- 个人设置中的待上线功能以灰色入口占位展示
- AI 语速三挡（舒适/标准/明快），全局设置在个人设置页面，各场域可局部覆盖，全局修改重置局部

## 反向补充：区域滚动行为规范

根据 demo 实际测试，各页面主内容区的行为：

| 页面 | 主内容区行为 | 说明 |
|------|-----------|------|
| 登录 | 固定全屏 | 内容自适配 |
| 大厅 | **可滚动** | Dashboard→课程→4库→社区纵向堆叠 |
| 授课 | **固定沉浸式** | PPT 填充视口，不滚动 |
| 对练 P1 情景导入 | **整体固定** | 内容自适配 |
| 对练 P2 角色扮演 | **外框固定、内部三列独立滚动** | Actor 背景/对话/Neo 辅导 各自滚动 |
| 对练 P3 反思复盘 | **外框固定、内部滚动** | 小结列表可长 |
| 学员报告 | **可滚动** | Markdown 长文档 |
| 调研 P1 说明 | **固定** | 内容短 |
| 调研 P2 沉浸对话 | **对话区滚动** | 随消息增加 |
| 调研 P3 信息确认 | **外框固定、内部滚动** | 信息列表可长 |
| 管理端（所有页） | **标题 sticky + 下方滚动** | 页面头部+CTA 固定顶部 |

**响应式**：所有页面适配 1280 / 1440 / 1920 宽度。
