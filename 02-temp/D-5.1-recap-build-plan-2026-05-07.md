# D-5.1 recap 相关 spec 构建规划

## 一、目录策略

### 方案 A · 局部增量（推荐）
```
00-public/product-V-0.2.0-D-5.1/        ← 新文件夹
├── README.md                            ← 说明 D-5.1 是 recap 增量版（其他 spec 仍以 D-5.0 为准）
├── VERSION.md                           ← D-5.1 版本元数据
├── CHANGELOG.md                         ← D-5.0 → D-5.1 仅 recap 相关变更
└── spec/
    ├── 00-glossary.md                   ← 仅 recap 相关术语增删（其他术语指向 D-5.0）
    ├── 05-learner/
    │   ├── 05-recap.md                  ★ 主文件 / 全量重写
    │   └── README.md                    ← 仅本目录说明
    └── 06-management/
        └── 03-report-center.md          ← 与 recap 联动的部分调整（如有需要）
```

**优点**：仓库不冗余 / 改动可见即变更 / 与 D-5.0 形成"基线 + delta"关系
**缺点**：D-5.1 不自包含 / 需读 D-5.0 才完整

### 方案 B · 全量复制
把 D-5.0 spec/ 22 文件全部复制到 D-5.1，然后改 recap 相关文件。
**优点**：D-5.1 自包含、可独立交付 / 与 D-5.0 完全解耦
**缺点**：22 文件 6 板块大量冗余 / 后续 D-5.0 修订需手动同步

**Claude 推荐方案 A**（理由：D-5.1 是 recap 局部迭代，不是全量重交付）

## 二、融合策略（D-5.0 + 4.5 决议 + v7 要点）

### 2.1 状态机简化（沿用 4.5 决议 1）

```
锁定（前序未完成 / 入口不可进）
   ↓ [所有关联 Activity 完成]
完整态首聊（一次性事件 / 完整 Walk-through + 应用计划）
   ↓ [Activity 完成事件触发]
自由复习态（永久 / 复习态 + 回访态二级切换）
```

**删**：D-5.0 的"空白态"+"部分态"（前序未完成时入口锁定即可，不展示半成品报告）
**Neo 行为态**：4 → 3（删"预览态"）

### 2.2 recap 定位（沿用 4.5 决议 2）

```
recap = 阶段性整合 + Neo 主动产出 + 跨 Activity 视角
```

与其他场域差异化：
- 不是 lecture 单次巩固 → recap 跨 Activity 整合
- 不是大厅辅导按需应用 → recap Neo 主动产出
- 不是 practice 单次复盘 → recap 阶段性多次行为整合

### 2.3 报告结构（融合 4.5 决议 3 + v7 要点）

**最终结构 = 5 章**（4.5 决议 4 块 + v7 综合评估开场章）：

| 章 | 标题 | 4.5 对应 | v7 对应 | 内容 |
|---|------|---------|---------|------|
| 第 1 章 | **综合评估** | — | 综合评估 | Neo 客观语气 1 段总结 + 跨课程引用 / **低数据降级为"本课程印象"** |
| 第 2 章 | **这一段我学了什么** | 决议 3 ① | 内容锚点 | 跨 Activity 知识 + 关键概念 / 工具 / 还有疑问 / **if-then 应用关联文字段（60-100 字）** |
| 第 3 章 | **这一段我表现怎么样** | 决议 3 ② | 学习表现 | 学习时长/节奏/频率 + 完成情况 + 高光/卡点（具体引用）+ **6 维画像** + **变化叙事**（Activity 起点 vs 终点 / 文字 + 对话片段举证） |
| 第 4 章 | **这一段我学得怎么样** | 决议 3 ③ | 学习成长 | 学习模式画像 + 学习节奏（深入 vs 浅尝）+ 状态变化（情绪/投入度）+ Neo 用人话解读（不直接给分数） |
| 第 5 章 | **我接下来该做什么** | 决议 3 ④ | 行动建议 | **1-3 条便签** + **立即/近期/中期** + 对应工作场景 + **Neo 任务卡片同步推送（学员选择接受）** |

**v6/v7 → 4.5 顺序变化**：
- v6/v7：综合 / 内容 / 成长 / 表现 / 行动
- D-5.1：综合 / 学了 / 表现 / 学得 / 接下来（**学得在表现之后**——先客观行为后主观推断）

**保留 v6/v7 5 章中的"综合评估"开场章**（作为 Neo 语气统领 + 跨课程记忆出场）—— 由 DM 决定是否保留

### 2.4 知识网络轻处理（沿用 4.5 决议 4）

- Neo **实时推导**跨 Activity 知识关联
- **不依赖 KGP 概念关系图**

- **不做流线/树状关联图**（v7 要点）→ 改为 **if-then 应用关联文字段**：「在 X 场景用 A 法 / 在 Y 场景用 B 法」60-100 字
