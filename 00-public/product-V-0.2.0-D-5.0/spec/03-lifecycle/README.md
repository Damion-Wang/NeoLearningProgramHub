## 关键概念

### 服务周期 vs 项目周期 二元锁定
- **服务周期** ≤ 12 月（客户购买周期）
- **项目周期** ≤ 服务周期 - 1 周（学习实际周期）
- 开营前 admin 可改 / 开营后 admin 锁定

### 项目里程碑（milestone）
- **阶段里程碑**：结营必选 + 中期可选
- **自定义里程碑**：上限 10 个 / 关联内容进度
- 触发：team report 草稿生成 + 站内前 3 天提醒 + Ora 当天提醒

### 数据生命周期 4 关键点
1. 项目隔离（每个项目独立数据包）
2. 服务周期内不清理
3. 到期前 1 月提醒
4. 归档永久原始保留

## 推荐阅读顺序

按生命周期顺序读：
1. **01-pre-learning** —— 项目怎么开始（含试运行 SOP + 学员名单 SOP）
2. **02-learning** —— 主战场（监控 / 报告 / 催学 / 配置）
3. **03-completion** —— 项目怎么结束（含数据交付）
