## 2. 渐进式加载体验

### 阶段一: 祝贺动画（3-5秒）

```
┌─────────────────────────────────────────────────────┐
│                                               [跳过] │
│                                                     │
│              ✨  恭喜你完成了                         │
│              ✨  横向协作！                           │
│              ✨                                      │
│              (粒子/光效动画)                          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

- 右上角"跳过"按钮，点击直接进入下一阶段
- 后台同时并行拉取碎片数据
- 性能约束: 动画帧率不低于30fps

### 阶段二: 预告文案

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│          正在为你汇总学习数据，生成专属报告...         │
│                                                     │
│          "正在回顾你的学习旅程..."                    │
│          → "正在分析你的能力成长..."                  │
│          → "正在整理行为证据..."                     │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 阶段三: 模块渐进呈现

模块按固定顺序渐入（每个间隔1-2秒）:

```
[已加载] ┌─ B1 高光时刻 ──────────────────┐
         │ 先来看看你的精彩瞬间             │
         │ Top3精选 + 行为证据卡片          │
         └─────────────────────────────────┘

[加载中] ┌─ B2 能力成长 ──────────────────┐
         │ 你的能力维度变化如何？            │
         │ [正在加载...]                   │
         └─────────────────────────────────┘

[等待中] ┌─ B3 认知转化 ──────────────────┐
         │ (骨架屏占位)                    │
         └─────────────────────────────────┘
```

**Agent行为:** 前2个模块加载完成时简短提示，后续不再重复避免打扰

### 阶段四: 加载完成

- 顶部提示"报告已完整生成"
- Agent: "你的学习报告已全部生成完毕。你可以浏览各个模块，也可以直接问我任何问题。"

### 吸顶导航

```
┌──────────────────────────────────────────────┐
│ 当前: 能力成长                                │
│ [高光] [能力] [认知] [表现] [技能] [成长] [综合] [建议] │
└──────────────────────────────────────────────┘
```

- 滚动时跟随当前可视模块自动切换
- 点击模块名快速跳转
- 模块名较长时自动缩略（移动端考虑折叠为下拉菜单）

### 关键交互

| 交互 | 触发 | 响应 | 联动 |
|------|------|------|------|
| 展开模块 | 点击摘要卡片 | 卡片展开详情+图表，其他模块下推 | Agent自动解读（安静模式关闭时） |
| 折叠模块 | 点击"收起"或标题 | 恢复摘要态 | 无 |
| 切换模块 | 展开态下点击另一模块 | 当前折叠+新模块展开 | Agent发送新模块解读 |
| 图表hover | 鼠标悬停图表数据点 | tooltip显示数值+时间+Activity名 | 无联动 |
| 图表点击 | 点击图表数据点 | 数据点高亮+Agent生成解读 | 节流: 前一个解读未完成时排队 |
| Agent引用 | Agent文本中标注数据 | 对应图表自动高亮5秒渐隐 | 如模块折叠则自动展开+滚动 |
| 安静模式 | 开启安静模式开关 | Agent不自动解读，仅回应主动提问 | 无 |
| 分位hover | 悬停分位标尺 | tooltip解释区间含义 | 无 |
| 分位点击 | 点击分位标尺 | Agent解读学员在该维度的相对位置 | 对话区新增解读 |

## 4. 双向联动

### 图表→Agent（用户主动）

```
用户点击雷达图"沟通能力"节点
    → 数据点高亮(颜色加深+放大)
    → 其他数据点降低透明度
    → Agent生成解读: "你在沟通能力维度的得分是78分..."
    → Agent引用行为证据(碎片ID检索)
```

### Agent→图表（Agent被动）

```
学员问"我的沟通能力怎么样？"
    → Agent回复包含[REF:chart=radar,dim=沟通能力]
    → 前端解析标注(学员不可见)
    → 对应图表模块自动展开(如折叠)
    → 数据点高亮脉冲1次, 5秒后渐隐
```

### 优先级规则
- 用户主动点击 > Agent被动引用
- 冲突时以用户最近操作为准，Agent联动静默

## 5. 报告讨论

### Agent脱敏机制

```
Agent回复中引用原始对话时:
  原始: "在对练中你说'我找市场部的张经理谈了一下'"
  脱敏: "在对练中你说'我找[***]谈了一下'"
  
自动检测并脱敏: 个人姓名(非角色名)、公司名、电话号码
```

### 质疑反馈闭环

```
学员: "我觉得这个分太低了"

Agent: "我理解你的想法。让我们一起看看对练中的
具体表现...[解释评分依据]...你的反馈已记录，
教学团队会在3个工作日内审核。如有调整会通知你。"
```

## 7. 匿名分位区间

### 正常展示（>=20人）

```
后25%           中间50%              前25%
░░░░░░░░░│████████████████████│●█████████
                                ↑ 你在这里
```

### 降级展示（<20人）

不展示分位标尺 → 文案"项目人数较少，暂不展示相对位置"

### 数据安全
- 学员端API: 仅返回分位区间标识
- HR端API: 不同认证域 + 返回数据含水印
- HR端常驻提醒: "排名数据仅供内部参考，请勿向学员透露"

## 9. 降级展示

| 级别 | 看板展示 | Agent说明 |
|------|---------|----------|
| 一级(部分数据) | 已完成模块正常+未完成模块"数据收集中..." | "部分模块数据还在加载中" |
| 二级(后台重试) | 保持"数据收集中..." | 无额外通知 |
| 三级(标注不可用) | "数据不足，暂无法展示"+鼓励文案 | "[模块名]暂时无法展示" |
| 四级(7天后工单) | "部分数据暂无法获取，技术团队正在处理" | 通知运营 |

## 10. 组件需求

| 组件 | 规格 |
|------|------|
| 祝贺动画 | 粒子/光效，>=30fps，"跳过"按钮 |
| 渐进加载指示 | 文案式进度（非百分比），骨架屏占位 |
| 模块摘要卡片 | 标题+1-2句结论+缩略图表，支持展开/折叠 |
| 交互式BI图表 | 雷达图/折线图/柱状图/饼图/热力图，支持hover/click/缩放 |
| 分位标尺 | 三段柱状标尺，学员位置小圆点，hover显示tooltip |
| 吸顶导航 | 8个模块名Tab，跟随滚动自动高亮，点击跳转 |
| 安静模式开关 | Toggle开关，位于对话区顶部 |
| 数据来源折叠区 | 默认折叠，展开显示Activity列表+碎片数+时间 |
| PDF导出按钮 | 两个位置（右上角+底部），支持loading态 |
| 行为证据卡片 | 碎片引用，包含来源Activity+引用原文（脱敏后） |
