## 2. 渐进式加载体验

> **[U8/D8修订]** 加载流程精简：祝贺动画缩为2秒；预告文案作为骨架屏上方标题合并展示；数据就绪的模块立即渲染，不人为延迟。

### 阶段一: 祝贺动画（2秒）

```
+-----------------------------------------------------+
|                                               [跳过] |
|                                                     |
|              恭喜你完成了                              |
|              横向协作！                               |
|              (粒子/光效动画)                           |
|                                                     |
+-----------------------------------------------------+
```

- 右上角"跳过"按钮，点击直接进入下一阶段
- 后台同时并行拉取碎片数据
- 性能约束: 动画帧率不低于30fps

### 阶段二: 渐进呈现

```
"正在为你汇总学习数据..." (骨架屏上方标题)

[已加载] +- B1 高光时刻 ----------------+
         | 先来看看你的精彩瞬间           |
         | Top3精选 + 行为证据卡片        |
         +-------------------------------+

[加载中] +- B2 能力成长 ----------------+
         | [骨架屏占位]                  |
         +-------------------------------+
```

**规则:** 数据就绪的模块立即渲染（不人为延迟），仅未就绪的显示骨架屏。

**Agent行为:** 前2个模块加载完成时简短提示，后续不再重复避免打扰。

### 阶段三: 加载完成

- 顶部提示"报告已完整生成"
- Agent: "你的学习报告已全部生成完毕。建议先看B1高光时刻。你可以浏览各个模块，也可以直接问我任何问题。"

### 吸顶导航

```
+----------------------------------------------+
| 当前: 能力成长                                |
| [高光] [能力] [认知] [表现] [技能] [成长] [综合] [建议] |
+----------------------------------------------+
```

- 滚动时跟随当前可视模块自动切换
- 点击模块名快速跳转

### 关键交互

| 交互 | 触发 | 响应 | 联动 |
|------|------|------|------|
| 展开模块 | 点击摘要卡片 | 卡片展开详情+图表，其他模块下推 | Agent自动解读（安静模式关闭时） |
| 折叠模块 | 点击"收起"或标题 | 恢复摘要态 | 无 |
| 切换模块 | 展开态下点击另一模块 | 当前折叠+新模块展开 | Agent发送新模块解读 |
| 图表hover | 鼠标悬停图表数据点 | tooltip显示数值+时间+Activity名 | 无联动 |
| 图表点击 | 点击图表数据点 | 数据点高亮+Agent生成解读 | **[X3修订]** 新点击取消前序排队中的解读请求，仅保留最后一次 |
| Agent引用 | Agent文本中标注数据 | **[X4修订]** 对话气泡中显示可点击"查看图表"链接，用户点击后才跳转展开对应模块 | 不自动滚动，避免打断阅读 |
| 安静模式 | 开启安静模式开关 | Agent不自动解读，仅回应主动提问 | 无 |
| 分位hover | 悬停分位标尺 | tooltip解释区间含义 | 无 |
| 分位点击 | 点击分位标尺 | Agent解读学员在该维度的相对位置 | 对话区新增解读 |

> **[U7修订]** 安静模式开关首次展示时增加tooltip: "开启后AI仅回答你的提问，不会主动解读"

## 4. 双向联动

### 图表->Agent（用户主动）

```
用户点击雷达图"沟通能力"节点
    -> 数据点高亮(颜色加深+放大)
    -> 其他数据点降低透明度
    -> Agent生成解读: "你在沟通能力维度的得分是78分..."
    -> Agent引用行为证据(碎片ID检索)
```

### Agent->图表（Agent被动）

```
学员问"我的沟通能力怎么样？"
    -> Agent回复包含[REF:chart=radar,dim=沟通能力]
    -> 后端解析层解析标注（非前端正则）
    -> 对话气泡中渲染可点击"查看图表"链接
    -> 用户点击 -> 对应图表模块自动展开+滚动
    -> 数据点高亮脉冲1次, 5秒后渐隐
```

> **[A4修订] 降级方案:** REF标注解析失败时，图表不联动，Agent文字仍正常显示，不影响阅读体验。

### 优先级规则
- 用户主动点击 > Agent被动引用
- 冲突时以用户最近操作为准，Agent联动静默

### MVP图表范围

> **[D5修订]** MVP阶段支持3种核心图表: 雷达图、折线图、柱状图。饼图和热力图降为P1迭代。

## 5. 报告讨论

### Agent脱敏机制

```
Agent回复中引用原始对话时:
  原始: "在对练中你说'我找市场部的张经理谈了一下'"
  脱敏: "在对练中你说'我找[***]谈了一下'"

自动检测并脱敏: 个人姓名(非角色名)、公司名、电话号码
```

> **[P6修订]** 脱敏仅针对学员真实输入中的PII，对练中的虚构角色名走白名单豁免，不脱敏。

### 质疑反馈闭环

```
学员: "我觉得这个分太低了"

Agent: "我理解你的想法。让我们一起看看对练中的
具体表现...[解释评分依据]...你的反馈已记录，
教学团队会在3个工作日内审核。如有调整会通知你。"
```

## 7. 匿名分位区间

### 正常展示（>=20人，且项目配置允许）

```
后25%           中间50%              前25%
ooooooooo|||||||||||||||||||||*|||||||||
                                ^ 你在这里
```

### 降级展示（<20人）

不展示分位标尺 -> 文案"项目人数较少，暂不展示相对位置"

### 数据安全
- 学员端API: 仅返回分位区间标识
- HR端API: 不同认证域 + 返回数据含水印
- HR端常驻提醒: "排名数据仅供内部参考，请勿向学员透露"

## 9. 降级展示

| 级别 | 看板展示 | Agent说明 |
|------|---------|----------|
| 一级(部分数据) | 已完成模块正常+未完成模块"数据收集中..." | "部分模块数据还在加载中" |

## 10. 组件需求

| 组件 | 规格 |
|------|------|
| 祝贺动画 | 粒子/光效，>=30fps，"跳过"按钮，2秒时长 |
| 渐进加载指示 | 文案式进度（非百分比），骨架屏占位，就绪即渲染 |
| 模块摘要卡片 | 标题+1-2句结论+缩略图表，支持展开/折叠 |
| 交互式BI图表 | MVP: 雷达图/折线图/柱状图，支持hover/click/缩放 |
| 分位标尺 | 三段柱状标尺，学员位置小圆点，hover显示tooltip，可配置是否展示 |
| 吸顶导航 | 8个模块名Tab，跟随滚动自动高亮，点击跳转 |
| 安静模式开关 | Toggle开关，位于对话区顶部，首次展示含tooltip |
| 数据来源折叠区 | 默认折叠，展开显示Activity列表+碎片数+时间 |
| PDF导出按钮 | 两个位置（右上角+底部），支持loading态 |
| 行为证据卡片 | 碎片引用，包含来源Activity+引用原文（脱敏后） |
| 图表引用链接 | 对话气泡中可点击的"查看图表"链接，点击后跳转对应模块 |

## 11. 报告版本关系

> **[P8/X5修订]** 学员端报告 = 只读快照。运营端报告工厂的审核标记不修改已发布版本；如需更正，生成新版本并通知学员"报告已更新"。已推送的报告不可撤回。
