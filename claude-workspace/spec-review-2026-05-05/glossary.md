## 4 · 学员端场域（统一口径 / DM 已决议）

| 术语 | 含义 | 备注 |
|---|---|---|
| **学习核心层 3 场域** | lecture + practice + recap | ★ **唯一统一口径** |
| 大厅 | 中枢层 / **不算场域** | 默认主页 / 与 3 场域并列描述 |
| ~~4 场域~~ | 旧口径 / 不再使用 | 全文修改 |
| ~~5 场域~~ | 旧口径 / 不再使用 | 含未来 inquiry/assignment / 全文修改 |
| 未来场域 | inquiry / assignment / sandbox（沙盘）等 | "未来会有新学习场域加入" |

**统一表述**：
- ✅ "学员端 = 大厅 + 3 学习场域"
- ✅ "学习核心层 3 场域（lecture / practice / recap）"
- ❌ "学员端 4 场域"
- ❌ "5 场域 × 7 动作矩阵"（应改为"3 场域 × 7 动作矩阵 / 含未来场域预留")
