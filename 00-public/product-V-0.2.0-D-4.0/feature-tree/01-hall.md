## 功能结构

```
A. Neo 在大厅的感知与决策（原 Leo 辅导能力，2026-04-26 起统一为 Neo）
   A1 Neo 在大厅的身份（Coach 能力，Program 全量知识）[ADR-003]
   A2 4 种人格模式（MENTOR / STRATEGIST / CHALLENGER / OBSERVER，自动无缝切换）
   A3 情绪感知（10 种情绪 + 4 类信号源 + 强负面自动 MENTOR）
   A4 8 种信号识别（Neo 在大厅处理 ⑤知行差距 / ⑦元认知偏差 / ⑧语境错配；①-④ ⑥ 由 Neo 在场域内处理）
   A5 5 种策略（TELL / ASK / REVIEW / PLAN / FRAMEWORK_GUIDED）
   A6 6 种对话阶段（OPENING→EXPLORING→DEEPENING→INSIGHT→ACTION→CLOSING）
   A7 角色优先级（辅导 > 课程推荐 > 工具笔记引导 > 督学）

B. Neo 在大厅的行动与产出
   B1 知识工具注入（关键词触发 + 渐进式披露检索）[ADR-006]
   B2 经验洞察捕捉（9 模式匹配 + 会话末批量提取 + 推荐笔记卡片）
   B3 行动计划检测（对话 → 看板待办）
   B4 课堂记忆引用（Bloom 层级策略 + 跨 Course 优先级 + 不暴露课程名）
   B5 复盘引导（5 级深度对应 Bloom）
   B6 认知链提问（同链不增加认知负荷）

C. Neo 在大厅的主动触发规则
   C1 首次登录（对话式自我介绍，不弹引导蒙层，需求感知推荐）
   C2 再次登录（欢迎回来 + 状态感知推荐，不硬推任务）
   C3 从场域返回（必然 1-2 句引用：学员先说话则延后，不省略）
   C4 学员主动提问（意图 + 信号 → 策略）
   C5 学员分享工作场景（触发经验洞察）
   C6 长时未学（温和提醒不施压）

D. 大厅布局
   D1 左侧导航区（个人页 / 社区置灰）
   D2 中间看板区（上下滚动长页面，主交互区）
   D3 右侧 Neo 对话（Chat UI）
   （三栏比例见 07.K2）

E. 学习看板组件
   E1 学习进度（进度条 + 阶段 + 累计时长 + 最近学习日期）
   E2 课程目录（S 型曲线 + hover Activity 列表 + ✅▶🔒 状态）
   E3 ★热力图（投入度 + hover 柱状图 + 时间/token 双指标）
   E4 ★工具库（逐步解锁 + 🔒未解锁标注 + PDF 下载）
   E5 笔记库入口（点击打开，详见 F 节）
   E6 待办任务
   E7 个人资产一键下载（ZIP 打包）
   E8 社区动态切片（本期 Constructing 占位）
   E9 ★个人画像面板（展示 AI 认知 · 学员修正入口 · 隐私仅自己可见 · 数据源见 09-A.B1）

F. 笔记（悬浮球 + 推荐卡片）
   F1 悬浮球全局常驻（大厅 + 所有场域）
   F2 展开悬浮窗（打字 + 粘贴 + 截图）
   F3 手动笔记（本期不与授课打通）
   F4 Neo 场域推荐卡片（授课场域高光瞬间，P3）
   F5 Neo 大厅推荐卡片（经验洞察捕捉，最多 3 条/会话，P3）
   F6 笔记库（查看 / 编辑 / 导出）

G. 场域进出与导航
   G1 AI 推荐进入（点击对话内任务卡片）
   G2 自主选择（左导航 / 看板目录）
   G3 自动衔接（上一场域完成后推荐下一个）
   G4 返回大厅（顶部悬浮按钮 + 0.3s 过渡）
   G5 场域间直达（不必回大厅）
   G6 ★自由复习（点击已完成 Activity 卡片 → Neo 答疑 + 翻看之前页面巩固）[spec 跨模块共享能力]
```

## 术语速查

- **渐进式披露**（ADR-006）= Neo 在大厅的知识库按文件层级逐层检索，不一次性注入所有内容
- **Bloom 层级** = 教育学认知分类（概念 / 理解 / 应用 / 迁移 等）
- **9 模式匹配** = 9 种经验碎片识别模式（工具应用 / 顿悟 / 反思 等）
- **认知链提问** = 围绕同一主题递进追问（澄清→深入→反思→假设→行动），同一链内不算新问题
