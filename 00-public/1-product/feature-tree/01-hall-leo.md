# 功能树 01：Hall + Leo（大厅与 Leo Coach）

**Owner：** 鼎明、显华
**日期：** 2026-04-21
**版本：** v0.4.0 对齐版（L1+L2）
**对应 Spec：** `spec/design/learner/01-hall.md` + `spec/design/learner/06-notes.md`

## 一句话定位

学员的家 + Leo Coach 智能体 + 悬浮球笔记。所有课堂外体验都在这里。

---

## 功能结构

```
A. Leo 角色能力
   A1 Leo 身份（Coach，Program 全量知识）[ADR-003]
   A2 4 种人格模式（MENTOR / STRATEGIST / CHALLENGER / OBSERVER，自动无缝切换）
   A3 情绪感知（10 种情绪 + 4 类信号源 + 强负面自动 MENTOR）
   A4 8 种信号识别（Leo 处理 ⑤知行差距 ⑦元认知偏差 ⑧语境错配，其余交 Neo）
   A5 5 种策略（TELL / ASK / REVIEW / PLAN / FRAMEWORK_GUIDED）
   A6 6 种对话阶段（OPENING→EXPLORING→DEEPENING→INSIGHT→ACTION→CLOSING）
   A7 角色优先级（辅导 > 课程推荐 > 工具笔记引导 > 督学）

B. Leo 核心技能
   B1 知识工具注入（关键词触发 + 渐进式披露检索）[ADR-006]
   B2 经验洞察捕捉（9 模式匹配 + 会话末批量提取 + 推荐笔记卡片）
   B3 行动计划检测（对话 → 看板待办）
   B4 课堂记忆引用（Bloom 层级策略 + 跨 Course 优先级 + 不暴露课程名）
   B5 复盘引导（5 级深度对应 Bloom）
   B6 认知链提问（同链不增加认知负荷）

C. Leo 主动触发规则
   C1 首次登录（LUI 自我介绍，不用 overlay，需求感知推荐）
   C2 再次登录（欢迎回来 + 状态感知推荐，不硬推任务）
   C3 从场域返回（必然 1-2 句引用，延迟不跳过）
   C4 学员主动提问（意图 + 信号 → 策略）
   C5 学员分享工作场景（触发经验洞察）
   C6 长时未学（温和提醒不施压）

D. 大厅布局
   D1 左侧导航区（个人页 / 社区置灰）
   D2 中间看板区（上下滚动长页面，主交互区）
   D3 右侧 Leo 对话（Chat UI）
   → 三栏比例定义在 Infra

E. 学习看板组件
   E1 学习进度（进度条 + 阶段 + 累计时长 + 最近学习日期）
   E2 课程目录（S 型曲线 + hover Activity 列表 + ✅▶🔒 状态）
   E3 ★热力图（投入度 + hover 柱状图 + 时间/token 双指标）
   E4 ★工具库（逐步解锁 + 🔒未解锁标注 + PDF 下载）
   E5 笔记库入口（跳 F 节）
   E6 待办任务
   E7 个人资产一键下载（ZIP 打包）
   E8 社区动态切片（本期 Constructing 占位）
   E9 ★个人画像面板 [补漏]
       E9.1 展示 AI 对我的认知（学习风格 / 兴趣领域 / 强项 / 待改进维度）
       E9.2 数据来源（Inquiry 画像采集 + Evaluation 学习特质碎片汇聚）
       E9.3 学员修正入口（对不准确的标签发起纠正，写入 DB 供后续迭代）
       E9.4 隐私与可见性（仅学员自己可见，HR / 运营不见个人画像详情）

F. Notes（笔记悬浮球 + 推荐卡片）
   F1 悬浮球全局常驻（大厅 + 所有场域）
   F2 展开悬浮窗（打字 + 粘贴 + 截图）
   F3 手动笔记（本期不与授课打通）
   F4 Neo 推荐卡片（授课场域高光瞬间，P1）
   F5 Leo 推荐卡片（经验洞察捕捉，最多 3 条/会话）
   F6 笔记库（查看 / 编辑 / 导出）

G. 场域进出与导航
   G1 AI 推荐进入（点击对话内任务卡片）
   G2 自主选择（左导航 / 看板目录）
   G3 自动衔接（上一场域完成后推荐下一个）
   G4 返回大厅（顶部悬浮按钮 + 0.3s 过渡）
   G5 场域间直达（不必回大厅）
```

## 与 v0.3.3 差异

- ★ **A 节完全重构**：0.3.3 仅"首次/非首次/日常辅导/记忆引用"四段笼统；v0.4.0 拆成人格/情绪/信号/策略/阶段 5 个正交维度（来自 coaching-skills 技术原型）
- ★ 新增 B 节 Leo 6 项核心技能（0.3.3 只提到"工具注入"）
- ★ E3 热力图 + E4 工具库解锁机制为 v0.4.0 新增
- ★ F 节 Notes 从大厅 BM4 提升为独立功能区（对应 learner/06-notes.md）
- 辅导教练正式命名为 **Leo**，类型明确为 Coach（不是 Tutor）
