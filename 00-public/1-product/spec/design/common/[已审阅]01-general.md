# 通用功能：账号管理 + 个人设置 + 帮助

## 模块定位

头像下拉菜单中的三个功能——账号管理、个人设置、帮助。点击菜单项后以**弹窗**形式打开，不跳离当前页面。全部纯 GUI，不需要 Agent。

---

## 一、账号管理页面

### 功能

| 功能 | 说明 |
|------|------|
| **个人信息查看** | 展示姓名、头像、手机号、邮箱等基本资料 |
| **个人信息编辑** | 支持修改姓名、上传/更换头像、绑定/更换手机号和邮箱 |
| **密码修改** | 输入旧密码 → 输入新密码 → 确认新密码 → 保存 |

### 用户故事

**张磊想修改自己的头像和密码**

1. 张磊点击 topbar 右侧头像 → 下拉菜单 → 点击"账号管理"
2. 弹窗打开账号管理，展示个人信息表单
3. 张磊点击头像区域的"更换"按钮，上传一张新照片，裁剪后保存
4. 张磊点击"修改密码"，输入旧密码和新密码，点击保存
5. 系统提示"密码修改成功"，下次登录生效
6. 张磊点击页面左上角返回按钮，回到之前的页面

### 已确认决策

- 本期不支持手机号自助注册（所有账号由运营导入创建）
- 对接 GPB 账户体系（登录验证由 GPB 处理）

## 二、个人设置页面

### 功能

| 功能 | 状态 | 说明 |
|------|------|------|
| **AI 语速** | ✅ 本期 | 1.0x ~ 1.5x 滑块调节。所有 1V1 对话场景统一应用，保持 AI 音色不变 |
| **页面风格** | ✅ 本期 | 浅色/深色模式切换 |
| **语种** | 🔒 待上线（置灰占位） | 界面语言切换 |
| **个人偏好** | 🔒 待上线（置灰占位） | 其他个性化偏好 |

### 用户故事

**张磊觉得 AI 老师说话太慢，想加速**

1. 张磊点击 topbar 头像 → 下拉菜单 → 点击"个人设置"
2. 弹窗打开个人设置
3. 张磊看到 AI 语速滑块，当前值 1.0x
4. 他把滑块拖到 1.2x，页面显示"已保存"
5. 返回授课教室，AI 老师的语速变快了，但音色没变
6. 张磊注意到下方有"语种""页面风格""个人偏好"三个灰色入口，标注"即将上线"

### 关键设计

- **语速在个人设置页面调整，不在播放界面**——避免学员以为这是视频课
- 调整后全局生效（授课/对练/辅导/测评所有对话场景）

## 三、帮助页面

### 功能

| 功能 | 说明 |
|------|------|
| **FAQ 列表** | 常见问题与解答（静态内容，本期手动维护） |
| **联系方式** | 客服/技术支持的联系方式 |

### 用户故事

**王芳不知道怎么查看自己的学习报告**

1. 王芳点击 topbar 头像 → 下拉菜单 → 点击"帮助"
2. 弹窗打开帮助，展示 FAQ 列表
3. 她在列表中找到"如何查看个人学习报告？"，点击展开答案
4. 答案说明了从大厅进入报告教室的路径
5. 王芳点击返回，回到之前的页面

### 本期范围

- 简单静态页面，FAQ 内容由运营手动维护
- 不做搜索功能
- 不做智能客服/AI 帮助

---

## 已确认决策（整体）

- 三个页面全部纯 GUI，无 Agent
- 从头像下拉菜单点击后以弹窗打开，关闭弹窗回到当前页面，不跳离
- 个人设置中的待上线功能以灰色入口占位展示
- AI 语速调整在个人设置页面而非播放界面
