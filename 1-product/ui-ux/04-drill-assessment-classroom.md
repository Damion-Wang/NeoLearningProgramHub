## 1. 阶段一: 情景导入

### 三栏内容定义
- **左侧目录区**: Project目录（无额外Tab）
- **中间看板区**: 三组信息卡片依次展示
- **右侧对话区**: 简要引导提示
- **默认比例**: 10:55:35

> **[X2修订]** 看板区顶部或导航条增加三步进度指示："导入 -> 演绎 -> 复盘"，当前阶段高亮。

### 信息卡片序列

```
第1屏: 背景介绍卡                    第2屏: 目标介绍卡                    第3屏: 技能介绍卡
+--------------------+    +--------------------+    +--------------------+
| 场景背景             |    | 对练目标             |    | 核心技能             |
|                    |    |                    |    |                    |
| 你的角色:           |    | 主目标:             |    | PIN法              |
| 项目经理张明        |    | 说服李红参与合作     |    | 立场-利益-需求分析   |
|                    |    |                    |    |                    |
| 对方角色:           |    | 分项目标:           |    | 关键提示:           |
| 市场部负责人李红     |    | 1. 理解真实顾虑     |    | 先找到对方的I(利益)  |
| 对跨部门合作有抵触   |    | 2. 找利益交叉点     |    | 不要急于表达P(立场)  |
|                    |    | 3. 达成行动方案     |    |                    |
|                    |    |                    |    |                    |
|      [下一步 ->]   |    |      [下一步 ->]   |    |    [开始对练 ->]    |
+--------------------+    +--------------------+    +--------------------+
```

> **[U3修订]** 二次进入（非首次对练同一Activity）时，三卡上方增加"快速跳过导入"按钮，点击直接进入演绎阶段。首次进入不显示此按钮。

**交互:** 点击"下一步"-> 当前卡片淡出(0.3秒)-> 下一组卡片淡入(0.3秒)

## 2. 阶段二: Roleplay演绎

### 三栏内容定义
- **左侧目录区**: Project目录
- **中间看板区**: AI角色形象 + 场景概览折叠面板
- **右侧对话区**: Roleplay多轮对话 + 学员主动结束按钮
- **默认比例**: 10:35:55

> **[U1/U2修订] 阶段切换动画分步串联:**
> 1. 导入卡片淡出(300ms)
> 2. 三栏比例平滑过渡到目标值(300ms)
> 3. 新内容淡入(300ms) + 角色形象渐现
> 三步串联而非同时变化，给用户明确的"场景已切换"感知锚点。

### 看板区布局

```
+-------------------------------+
|                               |
|      [AI角色形象: 李红]         |
|      (预生成的角色图)           |
|                               |
|      市场部负责人               |
|      "对合作持消极态度"         |
|                               |
+-------------------------------+
| > 场景概览 (折叠)              |
|   背景: ...                   |
|   目标: ...                   |
|   技能: ...                   |
+-------------------------------+
```

> **[D1修订]** 角色图在课程配置阶段预生成并缓存为静态资源，教室端仅加载图片URL，不触发实时文生图。

### 对话区布局

```
+--------------------------+
| [结束对练]  <- 右上角按钮 |
+--------------------------+
|                          |
| [李红]: 张明啊，你找我什么|
| 事？我现在真的忙得很。    |
|                          |
|         [张明]: 李红，我知|
|         道你最近很忙...   |
|                          |
| [李红]: 嗯，什么项目？    |
| 快说吧，我一会还有会。    |
|                          |
+--------------------------+
| [在这里输入回应...]  [发送]|
+--------------------------+
```

### 关键交互

| 交互 | 触发 | 响应 | 联动 |
|------|------|------|------|
| 对话交替 | 学员发送消息 | 内容安全过滤->导演分析->演员生成回应(首token 2秒内，流式输出) | 无 |
| 场景回看 | 点击"场景概览"折叠面板 | 展开显示背景/目标/技能信息 | 无 |
| 主动结束 | 点击"结束对练" | 确认弹窗->确定后导演生成结束语->进入复盘 | 三栏比例分步过渡 |
| 内容安全拦截 | 学员发送不当内容 | 提示"请围绕当前场景对话"，不传递给AI | 记录安全日志 |
| 催促 | 学员>3分钟未回应 | **[U4修订]** 第一次温和引导"需要时间思考没关系，准备好了告诉我"；第二次以角色语气催促 | 无 |

> **[A1/A2修订] 性能目标:**
> - 内容安全审核: <500ms响应，走独立安全服务异步检测+乐观放行
> - 对话响应: 首token 2秒内，流式输出（标注为目标值而非硬约束）

### 内容安全审核流

```
学员输入 -> 内容安全过滤层(<500ms) -> 通过? -> 是 -> 导演分析 -> 演员回应
                                       |
                                       +-- 否 -> 对话区提示"请围绕场景对话"
                                               + 记录安全事件日志
```

### 状态变化

| AI角色情绪 | 触发条件 | 视觉表现 |
|-----------|---------|---------|
| 冷淡 | 初始状态 | 短句、不耐烦语气 |
| 缓和 | 学员使用有效沟通策略 | 语气变软，开始展露需求 |
| 升级 | 学员施压/忽视 | 翻旧账，情绪对抗（不超天花板） |
| 合作 | 学员持续有效沟通 | 积极配合，讨论方案 |

### 降级/异常方案

> **[X14修订]** 对练教室降级方案:

| 异常场景 | 处理方式 |
|---------|---------|
| 网络中断 | 对话区提示"网络连接中断，尝试重连中..."，恢复后从断点继续 |
| AI响应超时(>10秒) | 对话区提示"AI思考中，请稍等..."；超过30秒提示"出了点问题，正在重试" |
| AI响应持续失败 | 提示"当前对练遇到技术问题，已自动保存断点"，引导返回大厅 |

## 3. 阶段三: 反思复盘

### 三栏内容定义
- **左侧目录区**: Project目录
- **中间看板区**: 对话回看面板（按轮次展示完整对话记录）
- **右侧对话区**: 教练身份的反思追问
- **默认比例**: 10:50:40（分步过渡同上）

### 看板区: 对话回看面板

```
+----------------------------------+
| 对话回看         [1][2][3]...[15] | <- 轮次导航
+----------------------------------+
|                                  |
| 第1轮 -----------------------    |
| [李红]: 张明啊，你找我什么事？    |
| [张明]: 李红，我知道你最近很忙... |
|                                  |
| 第2轮 -----------------------    |
| [李红]: 嗯，什么项目？           |
| [张明]: 是关于市场推广的合作...   |
|                                  |
| 第3轮 =======================    | <- 当前分析高亮（蓝色边框+浅蓝底色）
| [李红]: 又是CEO安排的？上次...   |
| [张明]: 这次不一样...            |
|                                  |
+----------------------------------+
```

> **[D2修订]** 轮次导航超过10轮时自动折叠中间轮次，如: [1][2][3]...[13][14][15]，可滚动查看。

### 对话区: 教练反思

```
角色切换动效: 李红头像 -> 0.5秒渐变 -> 教练图标
气泡颜色: 角色色 -> 渐变 -> 教练色

[教练]: 刚才的对话我都看了。你做得好的
地方是——第3轮你主动询问了李红的具体
困难，这是很好的共情策略。

[教练]: 在第5轮你提到了CEO的安排，
李红翻了旧账。你觉得为什么？

                     学员: 可能她觉得
                     我在用权力施压...

[教练]: 对的。那用PIN法分析，你跳过了
什么步骤？
```

### 关键交互

| 交互 | 触发 | 响应 | 联动 |
|------|------|------|------|
| 点击轮次 | 在回看面板点击某轮 | 该轮高亮 + 教练自动分析该轮 | 对话区新增分析消息 |
| 轮次导航 | 点击顶部数字按钮 | 面板快速滚动到该轮 | 无 |
| 教练分析联动 | 教练提到某轮 | 回看面板自动滚动并高亮该轮 | 同步联动 |
| 当前高亮 | 教练分析进行中 | 被分析轮次持续蓝色高亮 | 切换分析时高亮随之移动 |

## 4. 断点续播

> **[P1修订] 分阶段恢复策略:**

| 中断阶段 | 恢复行为 | 有效期 |
|---------|---------|--------|
| 阶段一(导入) | 从头开始导入（导入较短，无需保存进度） | 不限 |
| 阶段二(演绎) | 展示最后3轮摘要 -> AI从中断位置自然继续 | 7天 |
| 阶段三(复盘) | 恢复对话回看面板 + 教练从上次追问继续 | 7天 |
| 超过有效期 | 提示"距离上次对练已超过7天，建议重新开始" | - |

## 5. 对话式测评 (BEI/画像/困惑/高光)

### 三栏内容定义
- **左侧目录区**: Project目录
- **中间看板区**: 背景信息展示（测评主题+说明）
- **右侧对话区**: AI测评对话全程
- **默认比例**: 10:40:50

### BEI测评对话

```
AI: 接下来我会跟你聊大约10-15分钟，了解你的
一些工作经历。能分享一个你最近处理团队冲突的
真实案例吗？

                           学员: 上个月我们组有
                           两个人因为项目分工吵架了

AI: 当时是什么背景？涉及哪些人？(追问S)

                           学员: 我们做一个新项目...

AI: 你具体做了哪些动作？(追问A)

                           学员: 嗯...就一般处理了
                           (第1次模糊)

AI: 能具体说说"一般处理"包括哪些动作吗？

                           学员: 没什么特别的
                           (第2次模糊 -> 导演判断是否切换)

AI: 没关系，我们换个话题。能说说最近一次
你成功说服同事的经历吗？
```

> **[P2修订]** "2次模糊->切换话题"改为导演层综合判断是否切换，2次模糊仅作为建议阈值而非硬规则，导演可根据上下文灵活决定。

**时间预告:** AI开场必须包含"大约10-15分钟"的时间预告
**虚假案例标注:** 后台记录细节一致性评分，不向学员展示

## 7. 组件需求

| 组件 | 规格 |
|------|------|
| 信息卡片 | 背景/目标/技能三种卡片，淡入淡出切换，底部按钮；二次进入增加"快速跳过"按钮 |
| 角色形象展示 | 预生成角色图（静态资源），居中展示，形象下方角色名+描述 |
| 场景概览折叠面板 | 默认折叠，点击展开显示导入信息，在演绎阶段可随时回看 |
| 对话回看面板 | 按轮次展示，轮次导航条（>10轮折叠），高亮支持（蓝色边框+浅蓝底色） |
| 轮次导航 | 数字按钮横排，超10轮折叠中间，可快速跳转 |
| 三阶段进度条 | "导入->演绎->复盘"，当前阶段高亮，位于看板顶部或导航条 |
| 内容安全过滤提示 | 系统级提示条，与对话气泡视觉区分 |
| "结束对练"按钮 | 对话区右上角，次要样式，点击弹确认 |
| 量表GUI | 单选/多选/Likert/滑块，支持题目导航，进度指示 |
| 题目目录 | 已答(绿)/未答(灰)/当前(蓝)三态 |
| 角色切换渐变 | 0.5秒头像+颜色渐变动画 |
