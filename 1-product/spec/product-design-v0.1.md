### 跨模块依赖

- **碎片数据流**: 授课/对练/测评产出数据碎片 -> 报告模块消费聚合 -> 管理端展示
- **断点恢复**: 01框架定义弹窗规则，03/04各教室定义具体恢复策略
- **催学规则**: 06管理端HR与运营共用合并引擎，学员端02大厅消息中心展示
- **工具卡片**: 03授课教室产出 -> 02大厅汇聚展示
- **Agent统一**: 底层同一ChatPanel组件，通过模式配置适配9种Agent角色

## 3. 关键流程摘要

### 各模块核心流程

| 模块 | 核心流程 |
|------|---------|
| 01 框架 | 大厅<->教室切换 / 教室间直跳 / 断点恢复弹窗 / Agent上下文切换 |
| 02 大厅 | 首次引导 / 再次进入(>4h/<4h) / 进入教室 / 教室返回引用 / 催学消息退订 |
| 03 授课 | SCO调度推进 / PPT讲解+互动 / 视频播放+插嘴 / Quiz追问+提交 / 闭环反馈采集+复盘 |
| 04 对练 | 情景导入->Roleplay演绎->反思复盘 / 内容安全过滤 / BEI追问(STAR) / 量表测+评 |
| 05 报告 | 祝贺->渐进加载->浏览->讨论 / 图表<->Agent双向联动 / 质疑反馈闭环 / PDF导出 |
| 06 管理 | HR进度查看->Agent查询->报告生成 / 运营项目创建向导 / 报告工厂批量审核 / 催学协商 |

### 端到端学员旅程

```
首次登录 -> 大厅(教练自我介绍+任务卡片)
  -> 进入授课教室(PPT讲解+Quiz互动+闭环采集)
    -> 衔接卡片引导 -> 进入对练教室(导入->演绎->复盘)
      -> 进入测评教室(BEI对话/GUI量表)
        -> 返回大厅(教练引用消息+下一任务)
          -> 循环直到Course完成
            -> 进入报告教室(祝贺->报告浏览->讨论)
              -> 返回大厅(成就卡片+下一Course引导)
```

## 4. UX关键决策摘要

### 三栏布局规范

- **通用结构**: 左目录区 + 中看板区 + 右对话区
- **左栏折叠**: 点击图标或Ctrl+B折叠为24px图标条，中右按比例扩展
- **宽度拖拽**: 对话区左边缘可拖拽，session级保存

### 11种场景比例

| 场景 | 左:中:右 | 特征 |
|------|---------|------|
| 大厅 | 15:50:35 | 目录较宽，看板为主 |
| 授课PPT/对练导入/报告 | 10:55:35 | 内容展示为主 |
| 授课视频 | 10:65:25 | 视频区最大化 |
| 授课Quiz/测评对话 | 10:40:50 | 对话区为主 |
| 对练演绎 | 10:35:55 | 对话区占主导 |
| 对练复盘 | 10:50:40 | 看板回看+对话分析 |
| 测评量表 | 10:60:30 | GUI操作为主 |
| HR/运营端 | 15:60:25 | 管理操作为主，对话区精简 |

### 联动规则

- **看板->对话**: 点击元素 -> Agent 3秒内解读(2-3句)，500ms防抖+后到取消前
- **对话->看板**: Agent消息携带metadata -> 元素高亮5秒，同时最多1个
- **比例重置**: 不同类型/阶段切换重置为默认，同类型保持用户拖拽
- **断点弹窗**: <5分钟直接恢复，>=5分钟弹窗选择

### 组件清单(全局)

通用组件: 三栏布局容器 / ChatPanel(9种模式) / 骨架屏(4种模式) / Tab组件 / 折叠面板 / 过渡动画 / 悬浮导航条 / 拖拽手柄

学员端: 环形进度图 / 任务卡片 / 推荐卡片 / 待办列表 / 笔记卡片 / 工具卡片 / 消息列表 / PPT渲染器 / 视频播放器 / Quiz GUI(4题型) / 量表GUI / 对话回看面板 / 轮次导航 / BI图表(3种) / 分位标尺 / 吸顶导航 / 角色形象展示 / 信息卡片 / 三阶段进度条 / 安静模式开关

管理端: 项目选择器 / 进度表格(查看/管理两模式) / 半屏详情面板 / 催学规则表格 / 效果漏斗图 / 五步向导 / 报告预览 / 审核队列 / 批量进度条 / 体验模式提示条 / 数据高亮 / Agent催学预览 / 角色切换器 / 模板管理
