## 1. 大厅总览

### 信息架构

```
大厅页面（学员视角）
├── 左侧目录区 (15%)
│   ├── Course→Activity→SCO 三层折叠导航
│   │   ├── 每个Activity旁: 完成状态图标
│   │   └── 锁定Activity: 灰色+锁图标, hover显示前置条件
│   ├── 进度概览（总完成率百分比）
│   └── 消息入口（红点未读计数）
│
├── 中间看板区 (50%)
│   ├── BM1 学习进度（环形图+色块）
│   ├── BM2 课程推荐卡片
│   ├── BM6 待办任务列表
│   ├── BM4 笔记资料
│   ├── BM5 工具卡片（含来源标签）
│   └── BM3 消息中心
│
└── 右侧对话区 (35%)
    ├── 辅导教练头像+名称
    ├── 消息列表
    │   ├── 教练消息（左对齐气泡）
    │   ├── 学员消息（右对齐气泡）
    │   └── 嵌入式交互元素（任务卡片/知识卡片/确认按钮）
    └── 输入框 + 发送按钮
```

### 三栏内容定义

- **左侧目录区**: 课程结构导航，本期默认线性顺序解锁，每节点显示完成状态（绿勾/蓝进度条/灰圆点/灰锁）
- **中间看板区**: 学习仪表盘，固定顺序排列6个功能区域
- **右侧对话区**: 辅导教练Agent常驻，支持自然语言对话
- **默认比例**: 15:50:35

## 2. 左侧目录区

### 信息架构
```
课程目录
├── [展开] 横向协作 (Course) ─── 进度 67%
│   ├── ✅ PIN法（已完成）
│   ├── 🔵 55% BIC反馈法（进行中）
│   ├── ⚪ 谈判策略（未开始）
│   └── 🔒 综合对练（锁定: 请先完成谈判策略）
├── [折叠] 辅导反馈 (Course)
└── [折叠] 变革管理 (Course)
─────────────
📬 消息 (3)        ← 红点未读计数
```

### 关键交互

| 交互 | 触发 | 响应 | 联动 |
|------|------|------|------|
| 展开Course | 点击Course节点 | 0.2秒展开动画，显示子Activity列表 | 无 |
| 进入教室 | 点击可用Activity | 触发场景切换（→ux-01 2.1） | 三栏切换为教室模式 |
| 锁定提示 | hover锁定Activity | tooltip显示"请先完成[前置Activity名称]" | 无 |
| 点击消息 | 点击消息入口 | 看板自动滚动到消息中心区域 | 消息中心展开 |
| 折叠目录 | 点击折叠图标/Ctrl+B | 目录区折叠为图标条（24px） | 中+右按比例扩展 |

### 状态变化
| 状态 | 视觉表现 |
|------|---------|
| 已完成 | 绿色勾号 ✅ |
| 进行中 | 蓝色进度条 + 百分比 |
| 未开始 | 灰色圆点 ⚪ |
| 锁定 | 灰色 + 锁图标 🔒，不可点击 |

## 3. 中间看板区

### BM1 学习进度展示

**组件:** 环形进度图 + Course进度色块

```
┌─────────────────────────────────────────┐
│  学习进度                               │
│  ┌─────────┐                           │
│  │  67%    │  横向协作  ███████░░░ 67%  │
│  │ (环形图) │  辅导反馈  ████░░░░░░ 40%  │
│  │         │  变革管理  ░░░░░░░░░░  0%  │
│  └─────────┘                           │
│  [点击查看详情]                          │
└─────────────────────────────────────────┘
```

**交互:** 
- 点击环形图各段 → Agent自动解读（限2-3句话）
- 点击"查看详情" → 展开Course/Activity级细分进度
- 点击细分数据点 → 触发看板→对话联动

### BM2 课程推荐卡片

```
│  📚 推荐学习                            │

│  │ 授课 | 约45分钟 | 进行中 55%     │   │
│  │ 学习掌握BIC反馈三步法            │   │
│  │            [继续学习 →]          │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

**交互:** 点击卡片或"继续学习"按钮 → 进入对应教室

### BM6 待办任务列表

```
┌──────────────────────────┐
│ 🤖 小睿（辅导教练）        │
├──────────────────────────┤
│                          │

│  │                    │  │

│  │ 预计需要15小时完成   │  │
│  │                    │  │
│  │ 你的学习任务:        │  │
│  │ ┌────────────────┐ │  │
│  │ │📖 角色认知 45min│ │  │

│                          │
├──────────────────────────┤
│ [在这里输入消息...]  [发送] │

```

### 再次进入场景（>4小时间隔）

```
  ┌────────────────────┐
  │ 欢迎回来！          │
  │ 你上次完成了PIN法，  │
  │ 今天推荐学习BIC法。  │
  │ ┌────────────────┐ │
  │ │📖 BIC法  45min │ │
  │ │ 进行中55% [继续]│ │
  │ └────────────────┘ │
  └────────────────────┘
```

### 再次进入场景（<4小时间隔）

```
  ┌────────────────────┐
  │ 欢迎回来，有什么我能 │
  │ 帮你的？            │
  └────────────────────┘
```

### 嵌入式交互元素状态

| 状态 | 视觉 |
|------|------|
| 可点击（当前轮次） | 正常颜色，hover有高亮效果 |
| 已过期（学员已发新消息） | 灰色文字，不可点击，无hover效果 |

### 关键交互

| 交互 | 触发 | 响应 | 联动 |
|------|------|------|------|
| 文字发送 | 输入+Enter/点击发送 | 用户消息右对齐显示，Agent 3秒内开始流式回复 | 无 |
| 任务卡片点击 | 点击嵌入式任务卡片 | 触发场景切换进入教室 | 三栏切换 |
| 看板联动 | Agent提及实体 | 看板对应元素高亮5秒 | 同时最多1个高亮 |
| 教室返回引用 | 从教室回大厅 | 教练发送必然引用消息（1-2句话） | 无 |
| 宽度调整 | 拖拽左边缘 | 实时调整宽度，session级保存 | 看板区宽度反向联动 |

## 5. 状态变化总览

| 场景 | 首次进入 | 再次进入(>4h) | 再次进入(<4h) | 全部完成 | 从教室返回 |
|------|---------|-------------|-------------|---------|-----------|
| 教练消息 | 自我介绍+项目概况+任务卡片 | 进度+推荐+任务卡片 | 简化"欢迎回来" | 空闲引导 | 必然引用 |
| 看板高亮 | 无 | 无 | 无 | 笔记+工具区高亮 | 进度数据已更新 |
| 任务卡片 | 全部未开始 | 部分完成 | 同上次 | 成就卡片 | 刚完成的标记为完成 |

## 6. 组件需求

| 组件 | 规格 |
|------|------|
| 环形进度图 | 各段可点击，hover显示数值，支持动画 |
| 任务卡片 | 标题+类型图标+时长+进度+CTA按钮，嵌入对话气泡中 |
| 课程推荐卡片 | 大卡片，突出CTA |
| 待办列表 | 每行可点击，类型图标区分授课/对练/测评 |
| 笔记卡片 | 标题+来源(AI/手动)+日期，展开详情可编辑 |
| 工具卡片 | 缩略图+标题+来源标签+下载/分享图标 |
| 消息列表 | 标题+摘要+时间+已读未读，未读加粗+红点 |
| 对话气泡 | 左对齐(Agent)+右对齐(用户)，支持流式输出逐字显示 |
| 嵌入式按钮 | 在对话气泡中，支持enabled/disabled两态 |
| 三层折叠导航 | Course→Activity→SCO，每级缩进，状态图标 |
