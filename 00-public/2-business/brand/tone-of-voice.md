## 一、品牌性格特质（Brand Personality Traits）

睿学的品牌人格由四个特质构成。每一条都有明确边界——既要做到，也要避免过度。

### 1. 专业的（Professional）

> 让大型企业的HR负责人放心把培训项目交给我们。

- **含义：** 懂行业、懂客户、懂评估体系。我们说柯氏L3-L4不是在秀术语，而是因为这就是客户每天在纠结的问题。
- **边界：** 专业不等于学术化。不写论文，写解决方案。不堆叠缩略词，用客户听得懂的话讲清楚一件事。
- **对应定位：** 面向2000+人大型企业的HR/培训管理者，需要企业级信任感。 [Data]

### 2. 务实的（Honest / Evidence-based）

> 用数据说话，不用形容词说话。

- **含义：** 承诺可验证的效果，不承诺"颠覆行业"。说"行为改变数据"而非"全面赋能"。有数据就摆数据，没数据就说"我们正在验证"。
- **边界：** 务实不等于保守。我们对产品的信心是真实的，但表达时用证据而非口号。
- **对应定位：** 五大独特属性尚在验证阶段，对外材料使用"目标能力"而非"已实现能力"的措辞。 [Data]

### 3. 温暖的（Warm / Human-centered）

> 学习是人的事，不是系统的事。

- **含义：** 理解培训经理"两头受气"的处境，理解学员"被迫陪读"的无奈。我们的语气像一个经验丰富的同行，而非一台推销机器。
- **边界：** 温暖不等于煽情。不贩卖焦虑，不夸大痛苦。承认问题，然后给出路径。
- **对应定位：** 培训经理的核心困境是"政治生存问题"——我们是帮她解决问题的伙伴，不是旁观者。 [Data]

### 4. 前沿的（Innovative / AI-native）

> AI不是噱头，AI就是老师本身。

- **含义：** 我们是AI原生，不是在旧系统上贴AI标签。这个区别值得反复讲，但讲法是展示体验差异，而不是炫耀技术架构。
- **边界：** 前沿不等于炫技。客户不关心我们用了什么大模型，客户关心的是"培训效果能不能被证明"。技术是手段，效果是目的。
- **对应定位：** "AI原生 vs AI增强"是核心区隔，但需翻译为体验差异而非架构差异。 [Opinion]

## 二、语言原则（Voice Principles）

### "我们是 / 我们不是"对照表

| 我们是 | 我们不是 |
|--------|----------|
| **直接、有证据的。** 数据摆在前面，结论跟在后面。 | 模糊、堆砌概念的。不说"全方位赋能企业人才发展生态"。 |
| **温暖、鼓励的。** 理解客户的处境，站在同一边说话。 | 冷冰冰、居高临下的。不说"您的培训体系存在严重缺陷"。 |
| **诚实、有边界的。** 能做到的说能做到，还在验证的说还在验证。 | 过度承诺的。不说"彻底颠覆企业培训"或"100%提升培训效果"。 |
| **通俗、精确的。** 一句话说清楚一件事，不绕弯子。 | 术语轰炸的。不把"自适应学习"和"知识图谱"堆在一个句子里。 |
| **自信、从容的。** 知道自己的差异化在哪里，不急于证明。 | 焦虑、攻击性的。不贬低竞品，不制造恐慌。 |
| **尊重培训从业者的。** 赋能培训经理，让她更有话语权。 | 威胁培训从业者的。绝不暗示"AI会替代培训岗位"。 |
| **面向成年人的。** 假设读者有判断力，不需要被教育。 | 说教式的。不用"您知道吗？""据统计……"开头的陈腐句式。 |

## 三、写作样本（Writing Samples）

以下示例展示睿学品牌在不同场景下的具体语感。

### 3.1 官网首页标题

**推荐：**
> **AI就是老师。不是培训的附加功能，是培训本身。**
> 自适应教学 / 行为举证 / 自动化ROI报告——让培训经理第一次能用数据说话。

**备选：**
> **培训效果，终于看得见了。**
> 睿学——中国第一个AI原生培训交付平台。

**不要写成：**
> ~~赋能企业学习新生态，一站式AI智慧培训解决方案，助力组织人才发展数字化转型。~~

### 3.2 产品功能描述

**示例：自适应多路径教学**

> **千人千面，不再是口号。**
>
> 睿学实时诊断每位学员的掌握程度，动态调整教学路径。已经会的，跳过。没掌握的，换个方式再讲一遍。
>
> 63%的员工说培训内容与工作脱节——问题不是内容不好，是一刀切的交付方式让大多数人沦为"陪读"。睿学让每个人都学自己需要的内容。

**不要写成：**
> ~~基于先进的自适应学习引擎和知识图谱技术，平台能够智能化地为每位学员量身定制个性化学习路径，实现精准赋能。~~

### 3.3 错误提示 / 空状态

**示例：学习报告暂无数据**

> **还没有数据——但很快就会有。**
>
> 学员完成第一次AI对话后，行为数据就会自动出现在这里。不用手动统计，不用导出Excel。

**示例：网络连接异常**

> **网络不太稳定，睿学暂时上不了课。**
>
> 请检查网络连接后重试。已完成的学习进度不会丢失。

**不要写成：**
> ~~Error 500: Internal Server Error. Please contact administrator.~~
> ~~系统异常，请稍后重试。（错误码：E10042）~~

### 3.4 社交媒体帖子（微信公众号/知乎）

**示例：**

> 一个数字：82%的企业培训评估，止步于满意度问卷。
>
> 满意度高不代表学会了。考试通过不代表能用上。课后测试通过率80%，但实际工作应用不足30%。
>
> 从"感觉不错"到"行为改变"之间，缺的不是更多课程，是一种新的教学方式——AI能交互、能诊断、能举证的教学方式。
>
> #用数据说话 #睿学

**不要写成：**
> ~~重磅发布！睿学——颠覆性AI培训平台震撼上线！五大核心功能赋能企业人才发展新范式！~~

### 3.5 客户邮件（项目完成后）

**示例：**

> 张经理您好，
>
> 新员工入职培训项目已结束，附件是睿学自动生成的项目成果报告。几个关键数据供您参考：
>
> - 学员平均能力提升：从入职测评的52分到结项时的78分（Kirkpatrick L2-L3）
> - 高频错误模式：产品知识类问题集中在XX模块，建议后续针对性强化
> - 学员完成率：96%（AI交互模式下，无挂机/刷课情况）
>
> 如果您需要向管理层汇报，报告第三页的"行为改变摘要"可以直接使用。
>
> 有任何问题随时联系。

**不要写成：**
> ~~尊敬的客户，感谢您选择睿学平台。在贵公司的大力支持与全体学员的积极配合下，本次培训项目圆满完成……~~

### 3.6 销售演示开场白

**推荐：**

> "你们做培训评估的时候，汇报的数据通常有哪些？——出勤率、完成率、满意度问卷，对吧？这些数据有一个共同的问题：它们没办法回答'培训到底改变了员工的行为没有'这个问题。今天我想用15分钟给你看一个东西——睿学是怎么自动产出'行为改变'这一层数据的。"

**不要写成：**
> ~~"我先花5分钟介绍一下我们公司。绚星科技成立于……我们的愿景是……我们的睿学平台拥有八大核心功能……"~~

## 四、词汇指南（Vocabulary Guide）

### 4.1 优先使用的词汇

这些词来自客户真实语境，经目标受众研究验证，具有高共鸣度。 [Data]

| 优先词汇 | 使用场景 | 说明 |
|----------|----------|------|
| **千人千面** | 描述自适应教学能力 | 客户高频渴望词汇，直接对应痛点"千人一面" |
| **用数据说话** | 描述ROI报告和效果评估 | 客户原话，核心价值主张的通俗表达 |
| **学以致用** | 描述场景化教学和实战演练 | 学员视角的核心诉求 |
| **行为改变** | 描述Kirkpatrick L3级别评估 | 比"L3评估"更直觉，客户已有认知 |
| **走过场** | 描述当前培训现状（痛点） | 极高频痛点词汇，引发共鸣 |
| **看得见** | 描述效果可视化 | "效果看得见"比"效果可量化"更口语化 |
| **AI就是老师** | 描述AI原生定位 | 核心品牌叙事，一句话讲清区隔 |
| **实战演练** | 描述AI对练/情境模拟功能 | "不是考试，是实战" |
| **真成长** | 描述学习效果 | 与"走过场"形成对比 |

### 4.2 可以使用但需克制的词汇

| 词汇 | 限制条件 | 原因 |
|------|----------|------|
| **自适应** | 仅在专业语境（白皮书、技术文档）中使用，面向HR时用"千人千面" | 偏技术，非客户日常用语 |
| **Kirkpatrick / 柯氏模型** | 仅在HR已熟悉该框架时使用，否则翻译为"行为改变数据" | HR领域已有认知但非日常口语 |
| **AI原生** | 仅在定位和对比竞品时使用，不在每个句子里重复 | 容易变成空洞标签 |
| **降本增效** | 仅在定价和ROI叙事中使用，不作为核心价值主张 | 虽是客户语言，但太通用，不够差异化 |

### 4.3 禁止使用的词汇

| 禁用词汇 | 禁用原因 | 替代表达 | 标签 |
|----------|----------|----------|------|
| **赋能** | 中国企业软件圈最空洞的词，已丧失所有含义 | 具体说做了什么："帮你产出行为数据""让培训效果看得见" | [Opinion] |
| **颠覆** | 过度承诺，引发不信任；且暗示否定客户过去的工作 | "升级""重构""从XX到XX" | [Opinion] |
| **一站式** | 几乎所有SaaS都在用，完全没有区分度 | 具体说覆盖了哪些环节："教-学-练-测-报全链路" | [Opinion] |
| **智慧XX / 智能XX** | 2019年的营销词汇，已过时 | 直接说AI做了什么，不贴"智慧"标签 | [Opinion] |
| **生态** | 太宏大、太模糊，客户听了不知道你具体卖什么 | 说具体的产品能力 | [Opinion] |
| **替代培训师/替代HR** | 21.13%从业者担心被AI取代，触发防御心理 | "赋能培训经理""解放你的时间" | [Data] |
| **AI监控学习** | 触发隐私顾虑和员工抵触 | "AI教练""AI诊断" | [Opinion] |
| **最先进的AI / 大模型** | 技术导向，客户不关心底层技术 | "让培训效果看得见"——讲结果，不讲技术 | [Opinion] |
| **数字化转型** | 太宏大，且已被用滥 | 说具体的变化："从手工统计到自动报告""从视频课到AI交互" | [Opinion] |
| **全面/全方位** | 空洞修饰词，不增加任何信息量 | 删掉，直接说具体能力 | [Opinion] |

## Flags

### 🔴 Red Flags

1. **"赋能"一词在团队中根深蒂固。** 禁用"赋能"需要严格执行。建议在所有对外文案审核流程中加入词汇检查环节，发现"赋能""颠覆""一站式"等禁用词时直接打回修改。 [Opinion]

2. **品牌语言尚未经过客户测试。** 本文档中的写作样本和词汇偏好基于受众研究和定位策略推导，但目标客户是否真的对这种语气风格产生信任和好感尚需验证。建议在天使客户沟通中收集语言反馈。 [Assumption]

3. **"AI就是老师"表达需注意合规边界。** 在某些监管语境下（如教育行业相关政策），"AI就是老师"的表述可能引发敏感性。对外使用时应确认法务合规。 [Assumption]

### 🟡 Yellow Flags

1. **温暖与专业的平衡点因渠道而异。** 官网、销售演示偏专业/务实；社交媒体、社群运营偏温暖/通俗。不同渠道的语气权重需要在实际运营中微调。 [Opinion]

2. **禁用词汇清单可能引发内部阻力。** "赋能""一站式"等词在企业软件行业是默认表达，团队成员（尤其是销售）可能习惯性使用。需要配合内部培训和文案模板推行。 [Opinion]

3. **学员端语气尚未单独定义。** 本文档以面向HR/培训经理的B端沟通为主。睿学在学员端（App内对话、学习引导、反馈提示等）的语气需要单独定义——更口语化、更像教练、更少商业感。 [Assumption]

4. **竞品对比措辞需克制。** "马车装电机 vs 电动车"的比喻在销售场景有效，但在公开材料中可能显得攻击性过强。建议公开材料只讲自己的差异，不指名对比。 [Opinion]
